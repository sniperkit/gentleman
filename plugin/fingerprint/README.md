## 安装

### 二进制安装

请访问[版本发布页面](https://github.com/go-tgod/tgod/releases)下载编译好的程序

### 源码安装

安装前请使用`go version`命令确认你的Go环境版本大于等于 *1.8*

```Bash
go get -u github.com/go-tgod/tgod
```
